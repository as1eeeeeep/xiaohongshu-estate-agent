# 合规审核自检

## 不能写

| 类型 | 例子 |
|---|---|
| 收益承诺 | 稳赚、必涨、保值、包租、回报率保证 |
| 喊单词 | 闭眼买、神盘、真香、错过没了、赶紧上车 |
| 身份/升学承诺 | 买房拿身份、保证入学、学位稳了 |
| 极限词 | 最低、唯一、全网最低、必看第一 |
| 伪实拍 | AI 图暗示为真实房源现场 |

## 可以写，但要克制

| 表达 | 更稳写法 |
|---|---|
| 投资 | 配置、比较、流动性、租赁需求 |
| 升值 | 不写升值，改写价格锚点和成交对比 |
| 好租 | 租客来源更清楚，但不保证出租 |
| 捡漏 | 先看成交和估价，不用捡漏 |
| 名校 | 写校网/片区信息，不承诺入学 |

## 发布前 8 问

1. 这篇写给谁？
2. 第一眼有没有区域、价格或真实问题？
3. 有没有真实字段支撑？
4. 有没有至少一个风险？
5. 有没有编成交、租金、校网、地铁时间？
6. AI 图有没有伪装实拍？
7. 语气像真人，还是像汇报？
8. 私信引导是否自然，不制造焦虑？
