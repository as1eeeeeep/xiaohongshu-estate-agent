## 怎么学案例

不要只看成稿，要倒着看：

1. 这篇抓的目标人群是谁。
2. 封面第一眼给了什么理由。
3. 正文怎么把字段转成买方判断。
4. 哪个风险被放出来讲了。
5. 置顶评论怎么承接私信。
6. 哪些表达没有写，为什么不写。

## 好案例共性

| 共性 | 说明 |
|---|---|
| 数字具体 | 价格、面积、距离能追溯 |
| 语气不喊单 | 有判断，但不逼用户买 |
| 图片有分工 | 封面吸引，后页解释和举证 |
| 风险不躲 | 风险讲出来反而更像真人 |
| 留痕完整 | 后续能复盘为什么这么写 |
