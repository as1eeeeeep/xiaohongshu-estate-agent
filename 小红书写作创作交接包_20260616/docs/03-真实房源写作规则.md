# 真实房源写作规则

## 定位

账号口吻是买方视角的香港房产顾问，不是喊单中介。

## 写之前必须有

| 字段 | 要求 |
|---|---|
| 屋苑 / 区域 | 必须真实 |
| 价格 | 必须有来源和币种 |
| 实用面积 / 户型 | 必须有来源 |
| 地铁 / 生活圈 | 有真实距离或明确来源才写 |
| 成交锚点 | 有就写，没有就改成复核清单 |
| 风险 | 至少一个：楼龄、估价、维修、按揭、租务、政策 |

## 人群判断

一篇笔记只打一类主视角：

| 人群 | 内容重点 |
|---|---|
| 大陆投资型买家 | 流动性、租赁需求、价格锚点，不承诺回报 |
| 大陆家庭买家 | 生活圈、交通、长期持有，不承诺入学或身份 |
| 港漂上车客 | 总价、月供压力、通勤、实用性 |
| 跨境自住客 | 两地通勤、口岸动线、管理便利 |

## 字段怎么转成内容

| 原始字段 | 不要只写 | 应该写成 |
|---|---|---|
| `HK$440万` | 叫价 440 万 | 400 多万预算还能不能留港岛西 |
| `实用310呎` | 310 呎一房 | 面积小，重点看动线和收纳 |
| `西营盘站约500米` | 地铁近 | 生活圈和通勤成立，但别夸成无敌 |
| `20楼 / 一梯两户` | 楼层不错 | 同层少、安静感好，但不是决定性卖点 |
| `楼龄较高` | 老楼 | 估价、维修、按揭年期先核 |

## 正文底线

| 要有 | 不要 |
|---|---|
| 真实字段自然嵌入 | 一开头把字段堆满 |
| 买方判断 | 纯喊便宜、稀缺、值得 |
| 至少一个风险 | 把风险藏成小字 |
| 自然转化口 | 每篇都复制同一句私信话术 |
