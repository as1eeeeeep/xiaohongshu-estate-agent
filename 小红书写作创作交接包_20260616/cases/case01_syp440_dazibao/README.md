# 案例 01：SYP440 大字报封面

## 这个案例学什么

用生图模型直接做小红书大字报封面，而不是用本地硬叠一张信息卡。

## 成功点

| 点 | 做法 |
|---|---|
| 底图 | 真实房间全屏铺底，保留香港小户型真实感 |
| 标题 | 黄/白超粗中文，厚黑描边，硬阴影 |
| 主文案 | `400多万 / 还能留 / 港岛西？` |
| 事实贴纸 | `440万｜实用310呎`、`西营盘站约500米｜20楼` |
| 避开的坑 | 无底部黑条、无黑色价格栏、无软阴影高级感 |
