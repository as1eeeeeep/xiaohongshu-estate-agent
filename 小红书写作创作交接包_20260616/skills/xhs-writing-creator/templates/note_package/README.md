# 留痕包模板

每篇内容复制这个目录，改名为：

```text
YYYYMMDD-[主题]/
```

最少保留这些文件：

| 文件 | 用途 |
|---|---|
| `source_facts.md` | 原始事实和来源 |
| `sample_research.md` | 对标样本拆解 |
| `mimic_brief.md` | 学什么结构，不抄什么 |
| `title_options.md` | 标题公式和候选 |
| `draft.md` | 正文、标签、置顶评论 |
| `image_plan.md` | 每张图承担什么任务 |
| `imagegen_prompt.md` | 生图提示词和参考图 |
| `pre_publish_check.md` | 事实、合规、用户视角审核 |
| `final_note.md` | 最终发布稿 |
