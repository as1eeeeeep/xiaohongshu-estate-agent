# 小红书六类非房源内容 · 真实爆款帖子原型

以下是从小红书2025年港漂/香港买房相关真实帖子中提取的原型，
每种类型包含真实帖子的标题、正文结构、写作特征。
Agent4 请严格模仿这些原型的语气、节奏和结构，不要编造"中介救场"故事。

## 类型1：避坑指南

### 原型A：「香港买房避坑指南（血的教训总结）」
**作者**：小红薯6931CCB0 / 2025-09-23 / 103赞
**结构**：
- 开头：一个具体的惊险场景（差点签临约、被催定）
- 中段：列出3-5条具体坑，每条一小段，用自己或朋友的亲身经历
- 结尾：一句话总结+互动提问
**语气特征**：暴躁中带后怕，像在劝闺蜜，"血的教训""真的会破产"这种强烈情绪词

### 原型B：「港漂夫妇买奥运二手单位惨呻中伏！7个方法避免买到化妆楼」
**结构**：
- 开头：一句话讲结果（买到化装楼，地板踩出水）
- 中段：前业主怎么掩盖的（新油漆盖水痕、假天花藏漏水）
- 中段：自己的损失（维修要10万，前业主说"找我律师"）
- 结尾：列几条简明避坑法+求大家说说还遇到过什么坑
**语气特征**：真实愤怒中带无力感，"讲真"开头句很多，像是在跟朋友吐槽

## 类型2：区域生活测评

### 原型A：「搬来大围大半年 真实优缺点直白总结」
**结构**：
- 开头：一句话亮结论（优点缺点都很明显/不吹不黑）
- 中段：优点逐条讲（交通/物价/吃的），每一条带具体例子（菜心十几蚊、某种交通工具几点钟挤）
- 中段：缺点逐条讲（人多/环境旧/小强），不美化
- 结尾：总结什么人适合什么人别来 + 问同区人的感受
**语气特征**：像在写大众点评，优缺点三七开，有具体物价/时间/街名

### 原型B：「来港3年港漂女小红书大赞香港 连数11项优点」
**结构**：
- 开头：来港X年，三观被刷新
- 中段：列举式，每条一句话+一个具体场景
- 这种太像营销号的不适合我们，但要学习"一个观点配一个具体场景"的写法

## 类型3：买房全流程科普

### 原型A：「香港二手房实战血泪史：踩遍坑后，我把全过程总结成了这八步！」
**作者**：@W小姐和M先生的宅家日记
**结构**：
- 开头：一句话讲自己刚买完房/快累死了/趁热整理
- 中段：按时间线分步（预算→看房→谈价→律师→按揭→签约→验楼→收楼），每步2-3句，用大白话
- 重点：每一步都带一个自己踩过的坑/差点踩的坑
- 结尾：问大家走到哪一步了/有什么不懂的问我
**语气特征**：实用工具帖的口吻，"说实话""讲真"穿插，不是冷冰冰的教程

### 原型B：「终于在香港有属于自己的家啦」
**结构**：
- 开头：宣告上车（年龄+终于+房子）
- 中段：不是流程清单，而是心理历程（以为很简单→实操发现好复杂→咬牙搞定）
- 关键数字：首付多少、月供多少、省了哪些钱
- 结尾：鼓励还在看房的人
**语气特征**：像在朋友圈报喜，带emoji，情绪起伏明显

## 类型4：看房复盘日记

### 原型A：「暴走日出康城！看房看到腿断的一天」
**结构**：
- 开头：看了几套+身体感受（腿断/累瘫/头晕）
- 中段：每套房2-3句话，只讲最突出的一个点（好或烂都行）
  - 第1套：一个致命缺点直接劝退
  - 第2套：看着还行但有一个无语的点
  - 第3套：有点心动但还在纠结
- 中段：可以吐槽中介（走路太快/直男审美/泼冷水），但一句话带过不展开
- 结尾：还在纠结，求大家帮选/问大家看了多少套才定的
**语气特征**：朋友圈碎碎念，像在群聊吐槽，不需要完整故事线

### 原型B：「在荃湾连看三套房，我的腿已经快废了」
**结构**：
- 开头：周末看房特种兵+身体状态
- 中段：三套房的快速点评（唐楼爬五层→劝退 / 半新盘太小→憋屈 / 高层采光好→心动但临街）
- 关键：每套房只写一个最突出的记忆点，不要面面俱到
- 结尾：求问大家看了多少套才定的
**语气特征**：疲惫中带着幽默自嘲，像在跟朋友发语音

## 类型5：CitywalkHK探区

### 原型A：「离铜锣湾10分钟的静谧，大坑Citywalk」
**结构**：
- 开头：从哪里出发+第一印象（安静/慢/不像香港）
- 中段：具体写了什么（咖啡馆/街景/唐楼和新盘的对比），用具体街名和店名
- 中段：穿插一个居住相关的感受（路过一栋住宅，大堂很好看/楼下就是海）
- 结尾：感叹房价/自嘲买不起+问大家喜欢去哪逛
**语气特征**：松弛、有画面感、像在写游记但不是攻略腔
**重要**：这是"看房中途顺便逛"的Citywalk，不是纯粹旅游攻略。和"居住"有点关系但不硬推。

### 原型B：「假装在度假🌴坚尼地城这条海景街绝了」
**结构**：
- 开头：天气好+去了哪里
- 中段：海傍散步的细节（咖啡/叮叮车/遛狗的人）
- 中段：路过屋苑，楼下就是海，突然好想搬来
- 结尾：问有没有住这边的姐妹说说体验
**语气特征**：松弛、向往、带emoji和emoji表情，不写长段落

## 类型6：香港楼市行情评论

### 原型A：「美联储降息了，香港的房子现在能买吗？」
**结构**：
- 开头：看到新闻+群里炸了+我这个刚需有点慌
- 中段：自己的真实处境（首付攥手里、看了多久、预算多少）
- 中段：观察到的市场现象（什么盘在跌什么盘没跌），用的是自己的看房经历不是宏观数据
- 中段：问了懂行的人（中介/朋友/前辈），对方说了一句很实在的话
- 结尾：还在纠结+问大家一起讨论
**语气特征**：焦虑但理性的普通人视角，不装专家，"说实话""讲真"高频出现

### 原型B：「天天刷到香港大劈价，现在上车真的是接盘吗？」
**结构**：
- 开头：到处都是"大劈价""蚀让"的消息
- 中段：自己的观察（其实只有新界偏远盘在跳水，核心区根本没降）
- 中段：看了几套"便宜货"后发现要么楼层差要么朝向烂
- 结尾：手里预算越看越纠结，问大家觉得现在能捡漏还是都在半山腰
**语气特征**：怀疑论者视角，不轻易相信市场叙事，有独立思考


## 通用写作铁律（6种类型全部适用）

1. **用小红书真实用户的语气写，不要写成文章**
   - 像在微信给朋友发消息，不像在写公众号
   - 段落短（2-3句一段），多用换行
   - 可以有口头禅、可以碎碎念、可以情绪化

2. **必须带具体细节**
   - 一个具体数字（菜心十几蚊、走了两万步、月供差几百块）
   - 一个具体地名/街名/店名
   - 一个具体时间/频率（大半年/每个周末/看了三套）

3. **禁止任何形式的"中介救场"叙事**
   - 不能写：差点踩坑→中介发现→阻止→感恩
   - 不能写：中介拿出某个专业工具/数据
   - 中介可以出现，但只能是背景/吐槽对象/信息来源，不能是主角

4. **互动问题要自然**
   - 像在群聊里随口问的，不像在运营账号
   - "你们呢？""有没有人跟我一样？"比"欢迎评论交流"好一百倍

5. **标题要有搜索流量**
   - 带具体数字（7个方法/3套房/大半年/10万）
   - 带情绪词（差点破产/腿废了/说实话/太折磨了）
   - 带地域标签（香港/大围/荃湾/坚尼地城）
